# -Weboin-Technologies
=======
##  Weboin Creative Agency — Design Agency Homepage

Modern, responsive homepage built with **Next.js (App Router)** and **Tailwind CSS** for the Weboin Technologies Next.js developer internship task.

### Tech Stack

- **Next.js 16** (App Router, TypeScript)
- **React**
- **Tailwind CSS**
- **next-themes** (dark mode)
- **next/image** (image optimization)

### Features

- Hero, Services, Portfolio, and Contact sections
- Responsive layout (mobile-first)
- Dark / light mode toggle
- Basic form validation and success state
- SEO metadata via `metadata` in `layout.tsx`
- Hover animations on portfolio and service cards
